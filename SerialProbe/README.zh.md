# SerialProbe 串口调试工具

SerialProbe 是一款专为 macOS 设计的现代化串口调试工具，支持多语言（中英文）、丰富的串口参数配置、日志导出、快捷命令、自动刷新等功能。

## 功能特性
- 支持多种串口参数（波特率、数据位、校验位、停止位、流控）
- 快捷命令与常用指令一键发送
- 接收区与发送区日志导出
- 自动刷新串口设备列表
- 支持中英文界面切换
- 支持 macOS 深色模式

## 使用说明
1. 插入开发板，点击工具栏“刷新”扫描串口设备。
2. 选择端口和参数，点击“连接”建立串口通信。
3. 在发送区输入文本或 HEX 字节，点击“发送”即可发送数据。
4. 可通过“导出”按钮导出当前日志。
5. 支持快捷命令与常用指令一键发送。

## 国际化
- 默认支持简体中文和英文。
- 可根据系统语言自动切换界面。
