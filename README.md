# SerialProbe Serial Debug Tool

SerialProbe is a modern serial debugging tool designed for macOS, supporting multi-language (Chinese/English), rich serial parameter configuration, log export, quick commands, auto-refresh, and more.

## Features
- Supports various serial parameters (baud rate, data bits, parity, stop bits, flow control)
- One-click quick commands and instructions
- Export logs from receive and send panels
- Auto-refresh serial device list
- Supports Chinese and English UI
- macOS dark mode support

## Usage
1. Insert the development board and click "Refresh" in the toolbar to scan serial devices.
2. Select port and parameters, click "Connect" to establish serial communication.
3. Enter text or HEX bytes in the sender panel, click "Send" to transmit data.
4. Use the "Export" button to export current logs.
5. Quick commands and instructions can be sent with one click.

## Internationalization
- Supports Simplified Chinese and English by default.
- UI switches automatically according to system language.
